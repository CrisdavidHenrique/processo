Tarefa: Autorizar Projetos
Esta atividade têm o objetivo de autorizar a ordem para o início da execução de um projeto no NPI.
Disciplinas: Iniciar Projeto
Relacionamentos
FunçõesExecutor Primário:
  • Gerente de Projetos
Executores Adicionais:
Uso do Processo
  • Processo NPI  >  Iniciar Projeto  >  Autorizar Projetos
Etapas
1) Seleção de um Projeto candidato
Neste passo, é realizado a seleção dentre os projetos candidatos à execução (Tarefas definidas no Redmine como do tipo PROPOSTA), algum projeto que apresente características viáveis à sua execução (viabilidade).
2) Avaliação do Projeto Candidato
Após a seleção de um Projeto Candidato à execução, deve ser realizado a avaliação deste, com o propósito de verificar se o mesmo atende a um conjunto de requisitos obrigatórios. Estes requisitos foram definidos na Política de Gestão de Projetos; seu conteúdo pode ser visualiazado no ambiente Redmine na Issue 2097 (https://adm.quixada.ufc.br/issues/2097).
3) Liberação de Aval pelo Comitê de Projetos do NPI

O Comitê de Projetos do NPI é responsável pela Gestão do NPI; o mesmo pode ser formado por representantes dos Departamentos das Direções, Coordenações, Supervisores e Técnicos do Campus. A formação deste comitê é definida pela Política de Gestão de Projetos e pelo regimento do NPI.

Neste passo, o comitê deve liberar o aval para execução do projeto candidato, quando o projeto candidato apresentar as características necessárias a sua execução, julgadas pelo próprio comitê.

4) Autorização de Execução de Projeto
Uma vez que um projeto candidato foi avaliado e atende aos requisitos definidos e foi liberado pelo Comitê do NPI, o Gerente de Projetos poderá liberar a Ordem para execução do projeto. É necessário somente que a Situação da Tarefa referente ao Projeto Prospectado no REDMINE seja atualizada para EM ANDAMENTO.
