Artefato: Release do Produto
Este artefato tem como objetivo juntar todos os casos de uso implementados durante a Sprint e gerar uma release do produto.
Relacionamentos
FunçõesResponsável: Modificado Por:
  • Equipe de desenvolvimento
TarefasEntrada para:
  • Executar Teste
  • Realizar Reunião de Revisão da Sprint
Saída de:
  • Gerar Release
Uso do Processo
  • Processo NPI  >  Desenvolvimento  >  Sprint  >  Implementação  >  Release do Produto
  • Processo NPI  >  Desenvolvimento  >  Sprint  >  Testes  >  Release do Produto
  • Processo NPI  >  Desenvolvimento  >  Sprint  >  Encerrar Sprint  >  Release do Produto
