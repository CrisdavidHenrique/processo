Artefato: Documento de Especificação dos Requisitos
Neste artefato devem estar contidos todos os requisitos identificados durante o projeto.
Tipos de Produto de Trabalho: Produtos de Trabalho - NPI
Relacionamentos
FunçõesResponsável: Modificado Por:
  • Equipe de desenvolvimento
  • Líder Técnico
TarefasEntrada para:
  • Criar Backlog do Produto
Saída de:
  • Elicitar Requisitos
  • Validar Requisitos
Uso do Processo
  • Processo NPI  >  Desenvolvimento  >  Elicitação Inicial de Requisitos  >  Documento de Especificação dos Requisitos
Descrição
Descrição PrincipalSerá utilizado caso o nível de detalhamento do Documento de Visão não seja suficiente para definir o backlog do produto. O Documento de Visão pode já ser suficiente caso os requisitos descritos possam dar uma boa base para criar o backlog do produto.
